## Artmedia - Task

Built with React, JavaScript, CSS.
(animate.css, react-reveal, swiper, parallax-tilt)

## Installation and Setup Instructions

Installation:

`npm install`  

To Start Server:

`npm run dev`  

To Visit Web:

`http://localhost:5173/`
